Die Dealersoft wurde im engen Austausch mit Autohändlern entwickelt. Dem aufwändigen Entwicklungsprozess folgte eine technische und praktische Prüfung im Alltag mehrerer Autohändler:innen. Die JESAM Consulting GmbH, der Entwickler der Dealersoft, ist ein junges und dynamisches Team von Softwareingenieur:innen, das gemeinsam mit Expert:innen aus verschiedenen Bereichen professionelle Software entwickelt. Ziel der Entwicklung ist dabei immer, mithilfe der Software kurz-, mittel- und langfristig effiziente Leistungssteigerung zu erzielen - einfach, digital und für alle zugänglich.

Für wen ist Dealersoft geeignet? add remove

Dealersoft ist die Software für ambitionierte Autohändler. Verwalten Sie ab jetzt jederzeit und überall Ihre Fahrzeuge, Kund:innen, Kaufverträge und vieles mehr!

Dealersoft entlastet und beschleunigt alle Prozesse Ihres Autohandels - von Einkauf bis Verkauf, von Fahrzeugverwaltung bis Kaufvertrag.

 Welche Kaufverträge kann ich mit Dealersoft schreiben? add remove

Generieren Sie mit einem Klick eine fortlaufende Rechnungsnummer und erstellen Sie Ihre PDF Rechnung und Kaufvertrag. Egal ob Netto Export, $25A, inkl MwSt oder Stornovertrag, Dealersoft ist Ihr Tool für einen schnellen, einfachen Verkaufsprozess.

Kann ich von überall auf meinen Dealersoft Account zugreifen? add remove

Dealersoft ist eine Web und Mobile App basierte Software. Ihre Daten sind auf sicheren Servern gespeichert und können von Ihnen weltweit abgerufen werden.

Wie kann ich mein Passwort zurücksetzen? add remove

Mit der Passwort vergessen Funktion können Sie Ihr Passwort jederzeit zurücksetzen, diese finden Sie im Log im Bereich.
